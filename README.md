Система учёта заказов «Быстрая доставка»

Внутреннее приложение для автоматизации работы с клиентами и заказами. Поддерживает работу через графический интерфейс (GUI) и командную строку (CLI).

Стек технологий
Язык: Python 3.8+
База данных: SQLite
Интерфейс: Tkinter (GUI) / argparse (CLI)
Тестирование: pytest
